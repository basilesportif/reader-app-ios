# Reader App

A personal iOS app for querying book pages using vision-capable LLMs (Claude, OpenAI, Gemini).

## Structure

- `ios/` - iOS app (SwiftUI, iOS 17+)
- `backend/` - Future backend service (placeholder)

## Setup

1. Open `ios/ReaderApp.xcodeproj` in Xcode
2. Build and run on your device
3. Add API keys in Settings
